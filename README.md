## How These Were Made

Each analysis was generated by the `startup-competitors` skill, which dispatches parallel research agents to mine reviews, pricing, GTM signals, and more from real web data. The full process takes 15-25 minutes per market.

The skill runs three research waves — broad discovery, deep competitor profiling, and synthesis — then produces structured deliverables: a competitors report, competitive matrix, pricing landscape, and individual battle cards.

## Want to Analyze Your Own Market?

```bash
claude plugin marketplace add ferdinandobons/startup-skill
claude plugin install startup@startup-skill
```

Describe your market or competitors, and the skill runs automatically.
